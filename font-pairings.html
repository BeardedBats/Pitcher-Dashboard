Font Pairing Options

DM Sans (headers) paired with five secondary fonts for table data. Same real layout, different type feel.

Option 1 DM Sans + Inter

Inter is the workhorse of modern dashboards. Exceptional legibility at small sizes, tabular figure support, and a neutral personality that doesn't compete with DM Sans headers. font-family: 'Inter', sans-serif

Option 2 DM Sans + IBM Plex Mono

A refined monospace that gives stat tables a terminal/data-science feel. Every digit occupies the same width, so columns align perfectly without font-variant-numeric. Pairs beautifully with DM Sans' humanist warmth. font-family: 'IBM Plex Mono', monospace

Option 3 DM Sans + JetBrains Mono

Designed by JetBrains for developer tools, this mono has taller x-height and rounder letterforms than IBM Plex Mono — making it friendlier while keeping the data-dense precision. Great for a "sports analytics workstation" vibe. font-family: 'JetBrains Mono', monospace

Option 5 DM Sans + Space Mono

The boldest pick. Space Mono has geometric, almost retro-futuristic character shapes that lean into the "analytics command center" aesthetic. High contrast against DM Sans' smooth curves — this pairing has the most personality. font-family: 'Space Mono', monospace
